# BRAND shop

An online shop. It consists of 5 sample pages:
- the main page;
- the catalogue;
- one product description page;
- the cart;
- the checkout page;


### To run locally:

```
npm init --yes
npm i moment
npm i express
```

### With server:

```nodemon server/server.js```
